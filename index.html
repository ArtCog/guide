# Установка

Одна команда — и твой сотрудник нанят

Terminal 
curl -fsSL https://clawd.bot/install.sh | bash
PowerShell (от администратора) 
iwr -useb https://clawd.bot/install.ps1 | iex

Требования:

  • Node.js 22+ (установщик скачает автоматически)
  • API ключ Claude (Anthropic) или OpenAI
  • Telegram токен от @BotFather

# VPS: Запуск 24/7

Чтобы бот работал круглосуточно — поставь его на сервер

Рекомендуемый хостинг: Hetzner

  • Тариф: CX22 (~€4.5/мес)
  • 2 vCPU, 4 GB RAM, 40 GB SSD
  • Локация: Германия (Falkenstein)

Команды настройки:

1. Обновление системы 
sudo apt update && sudo apt upgrade -y
2. Установка Node.js 22 
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
3. Установка Clawdbot 
curl -fsSL https://clawd.bot/install.sh | bash
4. Запуск в фоне (после настройки) 
clawdbot onboard --install-daemon

Claude (Anthropic)

Рекомендуется для лучшего качества

OpenAI

Получить ключ

Оплата из России

Для оплаты API из РФ используй:

  • Виртуальные карты (например, через крипто-сервисы)
  • OpenRouter — агрегатор с альтернативной оплатой
  • Карты друзей/родственников за рубежом
