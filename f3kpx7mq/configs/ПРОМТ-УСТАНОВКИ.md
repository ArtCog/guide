# Установка конфигов ИИмерсивного — всё через Telegram

## Три действия

**1.** Из моего канала скачай четыре файла: `агенты-и-модели.json`, `безопасность.json`, `автономность.json` и этот промт.

**2.** Открой чат со своим OpenClaw-ботом и отправь ему первым сообщением текст промта ниже.

**3.** Следом отправь три JSON-файла как документы — прямо в Telegram. Бот прочитает их сам и начнёт задавать тебе вопросы.

---

## Текст промта (скопируй целиком)

```
Сейчас я отправлю тебе три JSON-файла от ИИмерсивного:
агенты-и-модели.json (база), безопасность.json (5 уровней защиты),
автономность.json (бот реально делает, не спрашивая на каждое действие).

Дождись все три файла. Затем:

1. Собери их через deep-merge — каждое поле живёт ровно в одном файле,
   конфликтов нет.

2. Прежде чем применять, задай мне одним сообщением все мои значения
   для полей REPLACE_ME_*:
   - Telegram ID (для allowlist и approvals)
   - Tavily API key (если нет — скажи где получить бесплатно)
   - Supadata API key (если использую скраппинг; иначе пропусти)
   - Email для Google OAuth и второго Codex-профиля
   - Tailscale hostname (иначе подставь localhost)
   - Хост биллинг-прокси (иначе удали секцию models.providers.anthropic.baseUrl)
   - Hermes MCP endpoint (иначе удали секцию mcp.servers.hermes)

3. Подставь мои ответы в соответствующие REPLACE_ME_* поля.

4. Коротко расскажи мне (без ИИ-воды):
   — какие способности я сейчас открываю (5-7 пунктов с примерами)
   — почему это сильный баланс (2-3 фишки, которые ИИмерсивный
     настраивал месяцами и я бы сам не догадался)
   — какие риски я принимаю и как их контролировать в Telegram

5. Сделай бэкап текущего openclaw.json. Покажи итоговый собранный
   JSON и спроси моего подтверждения до записи.

6. После записи проверь валидность через openclaw config validate
   и сообщи результат простыми словами.

Работай по шагам, на каждом давай мне видеть прогресс.
```

---

## Если что-то пошло не так

- **Не понял задачу** — отправь промт ещё раз, повторно в том же чате.
- **Ошибка при сборке или валидации** — напиши ему: «откати на бэкап и покажи что сломалось».
- **Совсем отказывается / циклится** — попробуй через другого агента (Claude Code или Codex), либо открой [guide.arturx.art](https://guide.arturx.art) и собери по шагам вручную.
- **Ничего не получается** — напиши ИИмерсивному в Telegram-канале, разберёмся.
