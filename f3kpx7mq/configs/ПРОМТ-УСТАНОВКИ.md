# Как установить конфиги ИИмерсивного за одну команду

Скачай три JSON-файла (`агенты-и-модели.json`, `безопасность.json`, `автономность.json`), положи их в любую папку на сервере, затем скопируй этот промт в чат со своим OpenClaw:

---

```
Вот три конфига от ИИмерсивного:
- агенты-и-модели.json (базовый стек агентов и моделей)
- безопасность.json (5 уровней защиты)
- автономность.json (бот реально делает, не спрашивая на каждое действие)

Что нужно сделать:

1. Прочитай все три файла через deep-merge.

2. ПЕРЕД тем, как применять — задай мне вопросы и собери мои значения
   для всех полей REPLACE_ME_*. Что именно нужно спросить:
   - Мой Telegram ID (для allowlist и approvals)
   - Мой Tavily API key (если есть; если нет — скажи что получить бесплатно)
   - Мой Supadata API key (если использую скраппинг; иначе можно пустую строку)
   - Мои email-адреса для Google OAuth и второго Codex-профиля (если нужны)
   - Мой Tailscale hostname (если использую Tailscale; иначе оставь localhost)
   - Мой IP/хост биллинг-прокси (если настроен; иначе удали секцию
     models.providers.anthropic.baseUrl)
   - Мой Hermes MCP endpoint (если использую Hermes; иначе удали mcp.servers.hermes)

   Спроси одним сообщением списком, а не по одному.

3. Подставь все мои значения в соответствующие REPLACE_ME_* поля.

4. ДО записи в мой openclaw.json — расскажи мне кратко и понятно (без ИИ-воды):
   - Какие конкретные способности я сейчас открываю (4 агента, автономия,
     экономия токенов, безопасность — в 5-7 пунктах с живыми примерами)
   - Почему это сильный баланс (ИИмерсивный настраивал эту систему несколько
     месяцев — перечисли 2-3 главные фишки, которые без него я бы не нашёл)
   - Какие риски я принимаю и как их контролировать (главные 2-3 риска +
     как их мониторить через логи и подтверждения в Telegram)

5. Покажи итоговый собранный openclaw.json и спроси моего подтверждения
   перед записью в /home/neroot/.openclaw/openclaw.json (или куда там у
   меня живёт конфиг).

6. После записи — проверь, что конфиг валидный через `openclaw config validate`
   и сообщи результат. Если есть варнинги — объясни их простыми словами.

Работай последовательно, на каждом шаге давай мне увидеть, что ты делаешь.
```

## Если что-то пошло не так

- **Конфиг не валидируется** — попроси OpenClaw показать ошибку и откатить на `openclaw.json.bak`
- **Бот не отвечает после перезапуска** — проверь логи: `journalctl -u openclaw-gateway -n 50`
- **Сабагенты не запускаются** — проверь, что Docker работает и образ собран (см. раздел "Субагенты" в гайде)

Все подробности по настройке — на guide.arturx.art в разделах, на которые ссылается каждый файл.
