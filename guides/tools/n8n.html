    n8n — визуальная автоматизация

    Self-hosted альтернатива Zapier. Визуальный редактор workflow: связывай API, базы данных и AI-агентов без написания кода.

    Что такое n8n?

    n8n (произносится «n-eight-n») — open-source платформа автоматизации с визуальным редактором. Ты перетаскиваешь блоки (ноды), соединяешь их — и получаешь рабочий pipeline.

    Зачем n8n, если есть Zapier / Make?

    ZapierMaken8n
    Self-hostedНетНетДа

    Быстрая установка

    Docker (рекомендуется)

    Terminal 
    docker run -d --name n8n \
  -p 5678:5678 \
  -v n8n_data:/home/node/.n8n \
  n8nio/n8n

    Откройте http://localhost:5678 — визуальный редактор готов.

    npm

    Terminal 
    npm install -g n8n
n8n start

    Что можно автоматизировать

    • Контент-машина — RSS → AI-обработка → Telegram/Twitter публикация
    • Мониторинг — webhook → проверка → уведомление в Slack
    • AI-агенты — LLM Chain + Vector Store + Tools = автономный агент
    • Обработка данных — Google Sheets → трансформация → PostgreSQL
    • Email-автоматизация — входящее письмо → классификация AI → ответ
