    Claude Code — AI в терминале

    CLI от Anthropic для работы с Claude прямо из консоли. Кодинг, рефакторинг, дебаг и управление проектами — без браузера и IDE.

    Что такое Claude Code?

    Claude Code — это официальный CLI (Command Line Interface) от Anthropic для работы с Claude. Вместо веб-интерфейса ты общаешься с AI прямо в терминале — там, где пишешь код.

    Ключевые возможности

    • Чтение и редактирование файлов — Claude видит твой проект и может вносить изменения
    • Выполнение команд — запуск тестов, билдов, git-операций
    • Контекст проекта — CLAUDE.md файлы хранят инструкции для AI
    • Безопасность — разрешения на каждое действие, sandbox режимы
    • Интеграция с IDE — работает в VS Code, JetBrains и любом терминале

    Чем отличается от ChatGPT / Claude.ai?

    Веб-интерфейсы работают с текстом. Claude Code работает с файлами. Он видит структуру проекта, читает код, запускает тесты и вносит правки — всё из терминала.

    Установка

    Terminal 
    npm install -g @anthropic-ai/claude-code

    Требования: Node.js 18+. После установки запустите claude в терминале и авторизуйтесь.

    Основы использования

    Terminal — примеры команд 
    # Запустить в текущей директории
claude

# Одноразовый вопрос
claude -p 'объясни что делает эта функция'

# Продолжить предыдущую сессию
claude --resume

    Claude Code автоматически читает CLAUDE.md в корне проекта — туда можно записать правила, стиль кода и контекст проекта.
