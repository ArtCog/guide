Claude API (Anthropic)

Пошаговое руководство по получению API ключа для Claude — одной из самых продвинутых AI-моделей. Нужен для ClawdBot и других интеграций.

Что такое Claude API?

Claude — это AI-модель от компании Anthropic, известная своей способностью к длинным рассуждениям и работе с большими контекстами (до 200K токенов).

API понадобится для:

  • ClawdBot — автономный AI-агент в Telegram
  • Создание чат-ботов и ассистентов
  • Анализ документов и кода
  • Автоматизация задач с AI
  • Интеграция в приложения
💡
Модели Claude

claude-3-opus — самая умная, дорогая
claude-3-sonnet — баланс цены и качества
claude-3-haiku — быстрая и дешёвая

Шаг 1: Создание аккаунта

1

Перейди на console.anthropic.com

Открой console.anthropic.com в браузере.

2

Зарегистрируйся

Можно использовать:

  • Email + пароль
  • Google аккаунт (быстрее)
3

Подтверди телефон

Anthropic требует SMS-верификацию. Введи реальный номер телефона — VoIP номера не принимаются.

⚠
Важно

Используй реальный номер телефона. Виртуальные номера (VoIP) обычно отклоняются системой.

Шаг 2: Панель управления

После регистрации ты попадёшь в Dashboard — главную панель управления. Здесь можно:

  • Создавать и управлять API ключами
  • Следить за использованием и расходами
  • Настраивать биллинг
  • Тестировать промпты в Workbench

Нажми кнопку "Start Building" чтобы перейти к созданию ключа.

Шаг 3: Создание API ключа

1

Открой раздел API Keys

В левом меню найди "API Keys" или перейди напрямую: console.anthropic.com/settings/keys

2

Нажми "+ Create Key"

Кнопка находится в правом верхнем углу.

3

Настрой ключ

Укажи:

  • Name — понятное название (например, "ClawdBot Production")
  • Workspace — рабочее пространство
  • Permissions — права доступа
4

Скопируй ключ

После создания ключ покажется только один раз. Сразу скопируй его в надёжное место.

🔒
Безопасность

API ключ показывается только один раз! Если потеряешь — придётся создавать новый. Никому не передавай ключ — он даёт полный доступ к твоему аккаунту и может списывать деньги.

Формат ключа:

sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Шаг 4: Настройка биллинга

Claude API работает по системе prepaid credits — ты пополняешь баланс, и с него списываются деньги за использование.

1

Перейди в Billing

В левом меню выбери "Billing" или "Plans".

2

Добавь платёжный метод

Принимаются банковские карты (Visa, Mastercard). Российские карты могут не работать — используй карту иностранного банка.

3

Пополни баланс

Рекомендую начать с $5-10 для тестирования. Этого хватит на тысячи запросов к Haiku или сотни к Sonnet.

💰
Бесплатный кредит

Новые пользователи часто получают бесплатные кредиты для тестирования. Проверь раздел Billing после регистрации.

Цены на модели

Claude API тарифицируется по количеству токенов (примерно 4 символа = 1 токен):

Модель Input (за 1M токенов) Output (за 1M токенов) Контекст
Claude 3 Opus $15 $75 200K
Claude 3 Sonnet $3 $15 200K
Claude 3 Haiku $0.25 $1.25 200K

Для ClawdBot рекомендую Claude 3 Sonnet — оптимальный баланс качества и цены.

Использование ключа

В ClawdBot

При настройке ClawdBot просто вставь ключ когда бот попросит:

ANTHROPIC_API_KEY=sk-ant-api03-xxxxx

В коде (Python)

import anthropic

client = anthropic.Anthropic(
    api_key="sk-ant-api03-xxxxx"
)

message = client.messages.create(
    model="claude-3-sonnet-20240229",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Привет, Claude!"}
    ]
)

print(message.content)

Через curl

curl https://api.anthropic.com/v1/messages \
  -H "x-api-key: sk-ant-api03-xxxxx" \
  -H "anthropic-version: 2023-06-01" \
  -H "content-type: application/json" \
  -d '{
    "model": "claude-3-sonnet-20240229",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'

FAQ

Не принимает мой номер телефона

Anthropic не принимает VoIP номера и некоторые операторы. Попробуй номер другого оператора или попроси друга с иностранным номером помочь с верификацией.

Российская карта не работает

Из-за санкций российские карты могут не приниматься. Варианты:

  • Карта казахстанского/армянского банка
  • Виртуальная карта (Wise, Revolut)
  • Криптокарта
Потерял API ключ, что делать?

Создай новый ключ в консоли и удали старый. Старый ключ нельзя восстановить, но можно отозвать для безопасности.

Как установить лимит расходов?

В разделе Billing → Usage Limits можно настроить:

  • Monthly spending limit — максимум в месяц
  • Rate limits — ограничения на количество запросов
Чем отличается от ChatGPT?

Claude и ChatGPT — разные компании и модели. Claude известен лучшей работой с длинными текстами и более "безопасными" ответами. Для ClawdBot используется именно Claude.
